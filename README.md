# EE Professora Nancy de Oliveira Fidalgo - Website

Este é o site institucional moderno da escola, desenvolvido com **Next.js**, **TypeScript** e **Tailwind CSS**.

## 🚀 Como substituir as imagens placeholder

Para deixar o site com a cara da escola, você deve substituir os arquivos na pasta `/public`.

### 1. Logo da Escola
- Substitua o arquivo `/public/logo-nancy.png` por uma versão PNG com fundo transparente.

### 2. Fotos da Galeria e Hero
Crie uma pasta chamada `imagens` dentro de `/public` e adicione os seguintes arquivos:
- `escola-hero.jpg`: Foto principal que aparece no topo do site (recomendado: 1920x1080px).
- `escola-sobre.jpg`: Foto que aparece ao lado do texto "História e Missão".
- `escola-1.jpg` até `escola-6.jpg`: Fotos para a galeria (recomendado: proporção 16:9).

### 3. Fotos do Grêmio
Crie uma pasta chamada `gremio` dentro de `/public` e adicione as fotos dos membros:
- `pres.jpg`, `vp.jpg`, `sec.jpg`, `tes.jpg`.

## 🛠️ Tecnologias Utilizadas
- **Next.js 16 (App Router)**
- **Tailwind CSS 4.0**
- **Lucide React** (Ícones)
- **Geist Font** (Tipografia moderna)

## 📱 Responsividade e Acessibilidade
O site foi construído seguindo as diretrizes WCAG:
- Menu hamburguer para dispositivos móveis.
- Contrastes validados.
- Tags ARIA para leitores de tela.
- Metadata completo para SEO e compartilhamento em redes sociais.
